# 🏋️ MARANNY

An AI-powered sports coaching platform that helps users discover, compare, and book verified private coaches through a modern cross-platform mobile application.

MARANNY integrates intelligent coach recommendations with Explainable AI (XAI) to improve transparency and help users make informed decisions when selecting sports coaches. The platform also supports booking management, real-time communication, notifications, and an equipment marketplace. :contentReference[oaicite:1]{index=1}

## ✨ Features

- 🔐 Secure Authentication
- 👤 User & Coach Profiles
- 🤖 AI Coach Recommendation System
- 🧠 Explainable AI (SHAP & LIME)
- 📅 Booking Management
- 💬 Real-Time Chat
- 🔔 Push Notifications
- ⭐ Ratings & Reviews
- 🛍 Sports Equipment Marketplace
- 🛡 Coach Verification
- 📊 Admin Dashboard
